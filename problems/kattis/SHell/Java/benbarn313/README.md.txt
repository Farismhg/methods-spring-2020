This is the Friendly Interactive SHell problem from Kattis. To use it you just run the Main.java file, insert how many commands you're going to be using, and then the commands themselves.
 
They whole point of the SHell is that when you insert a "^" charachter and it will autofill that ^ charachter with the command that was used as many lines ago as the number of ^'s you put down.

To run the program, run the main method in the Main.java file


Example usage:

Enter number of commands that will be entered, then enter the commands
-------------------------enter exit to quit-----------------------

IN: 3
OUT: 3

IN: you
OUT: you

IN: hello
OUT: hello

IN: hey ^^
OUT: hey you
