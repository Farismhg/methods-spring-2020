# FANTASY NAME GENERATOR	
Ever been in a fun role playing game, or perhaps a new video game and you need a new character name?
Well our awesome service has you covered. Just send in a request and our service will generate a snazzy new fantasy character name.

what our api does

## Instructions
The end point will be a simple website that has a NAME ME! button. The button will send a request to our back end which will
 generate a new unique character name. Then the name will be posted on the website.
